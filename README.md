# üé¨ MovieHub - Movie & TV Show Database

A modern, responsive web application for browsing movies and TV shows using the TMDB API. Built with vanilla JavaScript, HTML5, and CSS3 as a portfolio project.

![MovieHub Preview](preview.png)

## üåü Features

### Core Functionality
- **Real-time Search** - Search through millions of movies and TV shows
- **Multiple Categories** - Browse upcoming, popular, now playing, and top-rated content
- **Genre Filtering** - Filter content by your favorite genres
- **Advanced Filters** - Sort by popularity, rating, release date, and year
- **Favorites System** - Save your favorite movies and shows locally
- **Trailer Viewing** - Watch trailers directly in the app
- **Detailed Information** - View cast, ratings, runtime, and streaming availability
- **Smart Recommendations** - Get personalized content suggestions
- **View Modes** - Switch between grid and list layouts

### User Experience
- **Responsive Design** - Works seamlessly on desktop, tablet, and mobile
- **Keyboard Shortcuts** - Navigate efficiently with keyboard commands
- **Infinite Scroll** - Load more content as you scroll
- **Auto-scrolling Carousel** - Featured content carousel
- **Loading States** - Skeleton loaders for better perceived performance
- **Offline Detection** - Alerts when connection is lost
- **Toast Notifications** - User feedback for all actions
- **Lazy Loading** - Images load as they enter viewport

### Accessibility
- **ARIA Labels** - Proper accessibility attributes
- **Keyboard Navigation** - Full keyboard support
- **Focus Management** - Clear focus indicators
- **Reduced Motion** - Respects user motion preferences
- **High Contrast Support** - Better visibility options
